# Aircnc

<p align='justify'> SemanaOmnistack 9.0 idealizada pela Rocketseat com as aulas sendo ministradas por seu CTO Diego Fernandes
com objetivo de mostrar a comunidade de desenvolvedores as melhores tecnologias que são mais empregadas no mercado de Trabalho, 
sendo elas Node.JS no backend, ReactJS na web e React Native no mobile.</p>

<p align='justify'> A aplicação foi inspirada na empresa Airbnb onde <strong>"bnb"</strong> significa bed and breakfast,
logo a aplicação que será desenvolvida nessa semana, o Aircnc onde <strong>"cnc"</strong> significa code and coffee, onde as empresas
irão disponibilizar seus locais de trabalho para desenvolvedores que poderão conhecer o ambiente da empresa, receber mentoria ou até 
mesmo alugar espaços vagos além de outras funcionalidades.
</p>
